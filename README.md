# Phantom-Collector
PC : This is a vulnerability exploitation project aimed at bypassing modern EDR/AV solutions (Windows Defender 2025-2026). The project investigates architectural weaknesses in signature-based security and demonstrates the effectiveness of HTTP probing in a control and management (C2) environment. It helps to read all the data on the hard drive of another computer running Windows 11.
For educational and research purposes only.
